# Fine-tuning Llama-3-8B with QLoRA using Unsloth

Проект посвящен полному циклу дообучения языковой модели (LLM) под специфический стиль ответов (Instruction-following). 

## Технологический стек
* **Model:** Llama-3-8B-bnb-4bit
* **Method:** QLoRA (PEFT)
* **Frameworks:** Unsloth, Hugging Face Transformers, TRL
* **Hardware:** Google Colab T4 GPU

## Основные этапы работы

### 1. Подготовка данных
* Используемый датасет: `vicgalle/alpaca-gpt4`.
* Проведена фильтрация: удаление дубликатов, очистка коротких ответов (<20 символов).
* Итоговый объем: 200 примеров в формате JSONL.

### 2. Процесс обучения (Fine-tuning)
Обучение проводилось с использованием библиотеки **Unsloth**, что позволило эффективно использовать память T4 GPU.
* **LoRA Rank (r):** 16
* **Alpha:** 16
* **Steps:** 60
* **Learning Rate:** 2e-4
* **Optimizer:** AdamW 8-bit

| :--- | :--- |

#### Анализ выводов
* **Aligning:** Модель стала лучше соблюдать структуру ответов (списки, пошаговые инструкции).
* **Специфика:** Замечено небольшое переобучение на шаблонах разметки (попытки вставить ссылки на изображения там, где их нет).
* **Knowledge:** Базовые знания модели остались на прежнем уровне, но стиль стал более лаконичным и ориентированным на инструкцию.
